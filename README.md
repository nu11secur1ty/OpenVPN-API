### Install OpenVPN-API

1. enable PHP module 
```bash
a2enmod phpx.x
```
2. Enable CGI module
```bash
a2enmod cgi
```
3. Install Apache and make a Vhost
   
4. Download content into cgi-enabled folder in

- For UBUNTU
```bash
/var/www/html/cgi-enabled
```
5. Test

```url
HTTP://localhost/cgi-enabled
```
